## Overview

This project is a UI implementation exercise built from a Figma design file. The goal was to faithfully translate a visual design into clean, semantic HTML and CSS, including custom typography and a multi-column image gallery.

---

## Demo

Open `index.html` directly in your browser — no build step or server required.

## Design Reference

This project was implemented based on a Figma design. The focus areas include:

- Multi-column card/image grid layout
- Custom font pairing using Google Fonts
- Consistent spacing and sizing from the design spec
- Hover states and visual hierarchy

---

## Responsive Design

The layout adapts to different screen sizes using CSS media queries and flexible units, ensuring usability on both desktop and mobile devices.
